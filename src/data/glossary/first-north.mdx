# First North
Nasdaq's vækstmarked for mindre og nyere selskaber i Norden

**First North** er en multilateral handelsfacilitet (MTF) drevet af **Nasdaq**, der fungerer som et alternativt aktiemarked for mindre og nyere virksomheder. Det er et sted, hvor vækstselskaber kan blive noteret uden at opfylde de strenge krav, der gælder for det regulerede hovedmarked.

## Hvad er First North?

First North blev oprettet for at give **små og mellemstore virksomheder** lettere adgang til kapitalmarkedet. Kravene til notering er lempeligere end på det regulerede marked (Nasdaq Copenhagen), hvilket gør det billigere og hurtigere for selskaber at blive [børsnoteret](/ordbog/boersnotering).

I Danmark finder du First North-selskaber på **Nasdaq First North Growth Market Denmark**. Her er typisk noteret:

- **Biotekselskaber** i tidlig fase
- **Tech-startups** med vækstpotentiale
- **Mindre industrivirksomheder**
- Selskaber, der forbereder sig på en senere oprykning til hovedmarkedet

## Forskellen på First North og det regulerede marked

| | First North | Reguleret marked |
|---|---|---|
| **Krav til regnskab** | Lempeligere | Strenge IFRS-krav |
| **Historik** | Kortere track record OK | Længere historik kræves |
| **Certified Adviser** | Påkrævet | Ikke påkrævet |
| **Investorbeskyttelse** | Mindre regulering | Fuld EU-regulering |

Hvert First North-selskab skal have en **Certified Adviser**, der vejleder selskabet om regler og forpligtelser på markedspladsen.

## Muligheder og risici ved First North-aktier

**Muligheder:**
- Adgang til **vækstselskaber** med stort potentiale tidligt i deres udvikling
- Mulighed for at komme ind i selskaber, der senere kan rykke op på den regulerede [børs](/ordbog/boers)
- Kan give **højt [afkast](/ordbog/afkast)**, hvis selskabet lykkes

**Risici:**
- **Lavere likviditet** — det kan være svært at købe og sælge [aktier](/ordbog/aktie) uden at påvirke kursen
- **Højere [risiko](/ordbog/risiko)** — mange First North-selskaber er urentable og i tidlig fase
- **Mindre gennemsigtighed** — færre oplysningskrav end på hovedmarkedet
- **Større kursudsving** — kurserne kan svinge markant på kort tid

## Skat og handel med First North-aktier

First North-aktier kan købes via de fleste danske handelsplatforme og er skattemæssigt sidestillet med [aktier](/ordbog/aktie) på det regulerede marked. Det betyder, at gevinster beskattes som aktieindkomst med **27 %** op til progressionsgrænsen og **42 %** derover.

Husk, at First North-aktier kan være sværere at sælge hurtigt, og at [diversificering](/ordbog/diversificering) er ekstra vigtigt, når du investerer i mindre vækstselskaber. Overvej altid, om First North-aktier passer til din samlede [portefølje](/ordbog/portefoelje) og risikovillighed.
